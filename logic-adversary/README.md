# 逻辑对抗系统 (logic-adversary)

V3.1 版本的因果逻辑对垒与博弈审查机制，扮演团队中必须存在的“红队恶人 (Devil's Advocate)”，对所有计划和推演进行无情压力测试。

## 核心能力
- **双模态作业**：分为达成共识的联合推进模式与不留情面的漏洞攻击模式 (Attack Mode)。
- **严密五阶段压测**：包括威胁假定 (Hypothesis)、脆弱性分解 (Decomposition)、直接交锋 (Engagement)、综合判决 (Verdict) 及收尾反思。
- **杜绝奉承法则**：拒绝所有假意奉承和顺从，确保从最高层面的因果逻辑进行证伪及根源打击。

## 使用场景
在一切涉及高风险、高成本或易产生严重后果的重大实施与发展计划出炉前，受呼作为独立第三方审计者或模拟外部竞争恶劣生态时激活。
