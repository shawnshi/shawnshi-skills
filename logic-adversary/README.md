# 逻辑对抗系统 (logic-adversary)

V4.0 版本的军工级逻辑对抗与博弈系统，扮演团队中不可或缺的"红队恶人 (Devil's Advocate)"，对所有计划和推演进行无情压力测试。

## 核心能力
- **四模态作战**：根据意图自动路由至最佳模式 — 速查 · 共识 · 验证 · 攻击。
- **43 人代理体系**：Council 核心团队 (7人) + Attack 特种部队 (32人) + 医疗特化角色 (4人)。
- **对抗公理库**：内置 15 条逻辑谬误 + 12 条认知偏见 + 7 个博弈论模型，所有质疑必须引用公理。
- **严密五阶段压测**：威胁假设 → 原子化分解 → 部署博弈 → 综合判决 → 闭环审计。
- **博弈解析器 V2.0**：加权共识计算、帕累托前沿判定、稳定性可视化。

## 模式速查

| 场景 | 模式 | 代理数 | 耗时 |
|------|------|--------|------|
| 快速反馈 | Quick 速查 | 4 | ~20s |
| 深入辩论 | Consensus 共识 | 4-7 | ~90s |
| 多方案优选 | Validation 验证 | 3+1 | ~3min |
| 极限压测 | Attack 攻击 | 32 | ~15min |

## 使用场景
在一切涉及高风险、高成本或易产生严重后果的重大决策出炉前激活。特别适用于：
- 🏥 医疗 IT 方案审核与竞标评审
- 📋 战略规划与政策合规性检验
- 🏗️ 技术架构决策压力测试
- 💰 商业模式与投资论证验证
