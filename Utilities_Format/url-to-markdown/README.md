# 网络内容矿工 (url-to-markdown)

通过对 Chrome 开发者控制协议（CDP 引擎）进行底层驱动及控制，暴力提取并重构任意网页上的海量信息为其带来纯净极致转换体验的一环。

## 核心能力
- **抗动态变化的高清成像级解刨引擎**：专克含有极重型 JavaScript 后发渲染或是滚式加载（Lazy loading）逻辑的网页架构体系提取工作。
- **门禁锁突破与环境滞空停留模式 (Login Support)**：特别搭配能够呼出显性可见代理进程窗口，允许人工输入密码或验证身份后再交出截取指挥权的深度接驳能力模式 (`--wait`)。
- **去除一切虚妄废柴 (Smart Parsing)**：自带智能清洁滤网将对任何被认定为广告区块或不重要的站内侧边栏、尾部超链进行全面刮擦抹除操作，确保最终只有纯净的核心正文。

## 使用场景
当你向大语言模型提交一条被加密墙圈禁或极其臃肿的 URL，期盼模型能够去抓取剥去这些站点的杂质并且完整提炼消化原文大意、翻译等工作的时候触发使用。
