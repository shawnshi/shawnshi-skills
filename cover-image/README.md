## 核心功能
利用四维定制化参数（类型、风格、文字、情绪）实现出版级的视觉品牌包装。

## 战略契约
1. **备份逻辑**: 任何重新生成操作前必须执行旧版本归档，防止资产丢失。
2. **文字密度**: 严格遵守 8 字以内标题限制以确保视觉张力。
3. **风格锚定**: 封面风格必须与所属专栏的视觉规范（Visual Identity）严格一致。
