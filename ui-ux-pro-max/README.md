## 核心功能
全维度的 UI/UX 设计决策引擎。集成了 50+ 风格、97 种调色盘与 9 大技术栈规范，通过“主权+重写”模式实现跨页面的视觉一致性。

## 战略契约
1. **设计系统优先**: 任何 UI 实现前必须生成并持久化 `MASTER.md`，作为唯一的全局设计真相。
2. **零 Emoji 原则**: 严禁使用 Emoji 代替 UI 图标，必须使用 Heroicons 或 Lucide 等专业 SVG 矢量库。
3. **响应式审计**: 所有产出必须通过 Pre-Delivery Checklist，确保在 375px 至 1440px 范围内的交互鲁棒性。
