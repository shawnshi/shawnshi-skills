## 核心功能
作为实现前的硬性准入门槛，通过第一性原理分析强制执行技术选型与权衡分析。

## 战略契约
1. **问大于做**: 任何代码编写前必须产生设计文档，杜绝盲目编码。
2. **权衡优先**: 必须提供 2-3 种备选方案及其优劣分析（Trade-off Analysis）。
3. **分形自演化**: 讨论过程必须捕捉到潜在的 Skill 需求。
