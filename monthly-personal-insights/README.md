# 本月交互剖析仪 (monthly-personal-insights)

专门进行 30 日表现与用户 AI 交互模式审计分析的元分析引擎 (Meta-Analyst)，用以探知协作间的熵增模式并推动系统的长期进化。

## 核心能力
- **架构解耦高内聚**：实现了核心分析引擎 (`core/engine.py`) 与可视化表现层 (`assets/template.html`) 物理隔离，便于后续独立迭代。
- **活动采集与剥离**：使用自动化 Python 脚本深度挖掘并清洗全天候的会话记录、指令时长直至 Token 消耗分布。
- **六维因子切片分析**：将会话全息化转换为目标分类 (Goal)、满意度阈值表征 (Satisfaction)、摩擦发生概率 (Friction) 及最终成果性质判定。
- **交互式的战役级大屏仪表板生成**：自动汇总全月数据为本地 HTML 可视化报告，供决策及用户习惯修正参考。

## 使用场景
在定期需要盘点“当前用户或 AI 工作状态”、复盘分析使用量或寻求流程精益优化之时激活此自动化审计流水线。
