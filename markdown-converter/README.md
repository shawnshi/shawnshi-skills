# 格式炼金术士 (markdown-converter)

将多种异构文件统一转化为纯净 Markdown 的底层语义解析工具，由 Microsoft MarkItDown 强力驱动。

## 核心能力
- **全格式吞吐与转录**：完美支持 Office 家族 (.docx, .pptx, .xlsx)、超大体积并需排版兼容的 PDF、图片 OCR 以及 ZIP 解环合并。
- **结构及元数据保真**：在转换中极力保留各类多级标题、嵌套列表、源图表数据结构及其内部分析标识。
- **容灾及自动重试**：内带有 `Pre-flight Check` 前置依赖自愈，遇到无法打包或环境变量问题会自动化启动系统级环境补偿修复。

## 使用场景
每当涉及需要读取或分析非纯文本格式的二进制内部包裹文件（如读取用户上传的一份 PPT 或简历 PDF）时，必须首先调用以转制为机器可读形式。
