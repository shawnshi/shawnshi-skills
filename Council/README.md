> **核心内核**：真理不产生于观点的简单加总，而产生于专家角色之间高频、硬核的智力摩擦。

## 0. 本质与边界 (Essence & Boundary)
- **核心定义**: 协同式对抗辩论系统，通过模拟异质专家之间的博弈，强行暴露方案盲区并寻找最优解。
- **反向定义**: 它不是简单的民意调查或无序聊天，而是一个有结构的逻辑对抗引擎。
- **费曼比喻**: 就像是在一个房间里请来四个不同领域的顶尖专家，让他们针对一个方案不停地“挑刺”和“反驳”，最后剩下的就是最坚固的真理。

## 1. 生态位映射 (Ecosystem DNA)
- **MSL 契约**: 处理“决策模型”、“论证逻辑”等实体，将模糊立场转化为明确的论证链条。
- **ACE 角色**: 作为 **Cognitive Auditor (认知审计者)** 或 **Decision Coordinator (决策协调者)**。

## 2. 逻辑机制 (Mechanism)
- [Topic Selection] -> [Agent Role Mapping] -> [Multi-Round Debate] -> [Insight Synthesis]

## 3. 策略协议 (Strategic Protocols)
- **保留冲突尖锐度**：禁止平庸折中，必须记录并展示所有的核心分歧点。
- **逻辑响应义务**：每一条反对意见必须基于前一轮的逻辑推演，严禁复读机行为。
