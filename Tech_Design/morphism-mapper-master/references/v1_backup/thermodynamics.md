# Domain: 热力学
# Source: Clausius, Kelvin, Boltzmann, Gibbs
# Structural_Primitives: 能量、熵、温度、平衡、耗散

## Core Objects
- 系统: 被研究的物质集合，可以是开放、封闭或孤立
- 能量: 系统做功的能力，守恒但会降级
- 熵: 系统的无序程度，孤立系统总是趋向增加
- 温度: 热平衡的指标，能量传递的驱动力
- 热量: 由于温差而传递的能量
- 功: 有序的能量传递形式

## Core Morphisms
- 热传导: 热量从高温流向低温
- 做功: 能量转化为有序形式
- 耗散: 有序能量转化为无序热能
- 相变: 物质状态的突变，伴随潜热
- 热平衡: 系统趋向均匀温度状态
- 可逆过程: 理想化的无耗散过程

## Theorems / Patterns

### 1. 热力学第二定律
**内容**: 孤立系统的熵永不减少。能量总是从可用形式（有序）转化为不可用形式（无序）。

**Applicable_Structure**: 系统退化、秩序维持成本、效率极限、时间箭头

**Mapping_Hint**: 当 Domain A 面临系统混乱度增加时，认识到这是不可逆的自然趋势。问题不是"如何阻止"，而是"如何以最小代价管理"。

**Case_Study**: 热量总是从热物体流向冷物体，而不会自发反向流动。这定义了时间的方向。

### 2. 卡诺效率极限
**内容**: 热机效率存在理论极限，由高温热源和低温热源的温差决定。没有任何热机可以超过卡诺效率。

**Applicable_Structure**: 效率极限、资源转化、理想 vs 现实、边际收益递减

**Mapping_Hint**: 当 Domain A 追求效率提升时，先计算"卡诺极限"。接近极限时，投入产出的边际收益急剧下降，应考虑其他路径。

**Case_Study**: 蒸汽机的效率不可能无限提高，受限于锅炉温度和冷凝温度的比值。

### 3. 耗散结构
**内容**: 开放系统在远离平衡态时，通过与环境交换能量和物质，可以自发形成有序结构（耗散结构）。

**Applicable_Structure**: 开放系统、远离平衡、自组织、动态平衡

**Mapping_Hint**: 当 Domain A 是封闭系统且趋向混乱时，考虑开放边界，引入能量/物质/信息的流动，可能自发产生新的有序结构。

**Case_Study**: 贝纳尔对流：从下方加热的液体薄层，在特定条件下自发形成规则的六边形对流胞。

### 4. 麦克斯韦妖
**内容**: 一个假想的"妖"可以观察分子运动，只让快分子通过，从而违反热力学第二定律。但信息擦除需要能量，妖本身产生熵。

**Applicable_Structure**: 信息 vs 能量、观察的成本、控制的代价、信息熵

**Mapping_Hint**: 当 Domain A 试图通过"智能筛选"提高效率时，考虑信息处理的成本。获取和处理信息本身需要消耗资源。

**Case_Study**: 麦克斯韦设想的妖可以分离快慢分子，但兰道尔原理证明擦除信息需要能量，妖无法违反热力学第二定律。

### 5. 相变与临界现象
**内容**: 系统在临界点附近会表现出剧烈的性质变化，微小的扰动可以引发宏观效应。

**Applicable_Structure**: 临界点、相变、杠杆效应、非线性响应

**Mapping_Hint**: 当 Domain A 处于稳定状态时，寻找可能的"相变点"。在临界点附近，微小的能量投入可能引发系统性质的质变。

**Case_Study**: 水在0°C时从液态变为固态，温度微小的变化导致性质的巨大改变。临界乳光现象显示系统在临界点的敏感性。
