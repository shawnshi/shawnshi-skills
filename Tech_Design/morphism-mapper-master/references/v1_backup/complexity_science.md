# Domain: Complexity Science
# Source: Murray Gell-Mann, Stuart Kauffman, John Holland, Per Bak
# Structural_Primitives: emergence, self_organization, chaos, adaptivity, fitness_landscape

## Core Objects
- **Agent**: 具有局部信息、遵循简单规则的自主行为体
- **Emergent Pattern**: 系统整体表现出的、个体不具备的宏观行为
- **Fitness Landscape**: 适应度的多维地形，决定进化的方向与速度
- **Attractor**: 系统动态演化的稳定终态或周期性轨道
- **Phase Transition**: 系统从一种宏观状态到另一种的突变临界点

## Core Morphisms
- **Interaction**: Agent之间的信息/能量/物质交换，产生非线性效应
- **Feedback**: 输出作为输入影响系统，正负反馈决定稳定性
- **Selection**: 基于适应度的优胜劣汰，塑造群体特征
- **Mutation/Variation**: 引入新变异，维持系统探索能力
- **Aggregation**: 微观行为涌现宏观模式（如鸟群、蚁群）

## Theorems / Patterns

### 1. Edge of Chaos
**内容**: 系统在秩序与混沌的边界（相变临界点）具有最高的计算能力和适应潜力

**Applicable_Structure**: 需要平衡稳定性与创新的复杂系统

**Mapping_Hint**: 映射到"组织管理的松紧平衡"、"市场周期的投资时机"、"产品稳定与迭代的节奏"

**Case_Study**: 基因调控网络处于临界状态，既能维持细胞功能又能响应环境变化

### 2. Power Law Distribution
**内容**: 复杂系统中，事件规模与发生频率呈幂律关系，小事件频繁，大事件罕见但影响巨大

**Applicable_Structure**: 具有尺度不变性的统计分布（地震、股市崩盘、流行病传播）

**Mapping_Hint**: 映射到"产品使用分布"、"城市规模分布"、"财富分配"、"黑天鹅事件的必然性"

**Case_Study**: 地震频率服从幂律（古腾堡-里希特定律），无法精确预测但可评估长期风险

### 3. Self-Organized Criticality (SOC)
**内容**: 系统自发演化到临界状态，无需外部参数调节，小扰动可能引发级联反应

**Applicable_Structure**: 具有内在动态平衡的复杂系统

**Mapping_Hint**: 映射到"市场泡沫的形成与破裂"、"舆论引爆点"、"供应链的级联故障"

**Case_Study**: 沙堆模型中，沙粒不断添加使系统自组织到临界状态，引发规模不一的崩塌

### 4. Fitness Landscape Dynamics
**内容**: 适应度地形随进化过程改变（红皇后效应），局部最优可能随时间变为陷阱

**Applicable_Structure**: 多主体共同进化的竞争环境

**Mapping_Hint**: 映射到"颠覆性创新打破行业格局"、"动态竞争中的持续投资必要性"、"平台化转型的时机"

**Case_Study**: 科技行业中，昨日的技术领先（如诺基亚的功能机）可能成为明日负担

### 5. Path Dependence
**内容**: 系统的历史轨迹影响未来演化，早期小事件可能锁定长期发展路径

**Applicable_Structure**: 具有累积效应和历史锁定现象的系统

**Mapping_Hint**: 映射到"技术标准竞争"、"品牌定位的初始选择"、"组织文化的形成"、"沉没成本的困境"

**Case_Study**: QWERTY键盘布局因早期打字机机械限制而流行，尽管效率并非最优
