# 多智能体作家 (multi-agent-writer)

顶级咨询级协作写作编排专家 (V9.0: The Automated Strategic Engine)。融合麦肯锡金字塔原理、Ghost Deck 视觉推演与多角色红队博弈。自动集成 `humanizer-zh-pro` 进行极致的去 AI 化清洗。

## 核心能力
- 强制实施结论先行 (Answer-First)、叙事流转 (Narrative Flow)、动词驱动 (Verb-Driven) 和心跳节奏 (Heartbeat Rhythm)。
- **Phase 0：战略对齐**：锁定受众、议题与反共识目标，强制使用搜索工具获取真实数据锚点。
- **Phase 1：红队圆桌会议**：通过主题专家、恶魔辩护人与合伙人三角色博弈，提炼 3-5 个经得起残酷推敲的核心支柱论点。
- **Phase 2：Ghost Deck 逻辑设计**：在正式落笔前构建极简骨架，强制使用 Action Titles 与视觉逻辑编排。
- **Phase 3：散文体起草**：严格遵循金字塔结构与 MECE 原则，起草高信噪比的战略散文。
- **Phase 4：逻辑审计**：对照 17 项检查清单与 33+ 条反模式库进行全量扫描，执行 3D 逻辑评分。
- **Phase 5：最终交付**：附带执行摘要与红队残留风险附录。

## 使用场景
用于撰写高影响力的战略备忘录、董事会级汇报、深度行业长文或任何要求极高认知清晰度和抗打压逻辑的文档工作时。
