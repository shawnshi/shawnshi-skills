## 核心功能
咨询级协作写作工作流。强制执行“金字塔原理”与“结论标题化”，通过多智能体博弈确保输出的每一段文字都具备战略清晰度。

## 战略契约
1. **结论先行**: 文章大纲必须由“行动标题” (Action Titles) 组成，确保即便只读标题也能完成逻辑决策。
2. **视觉驱动**: 强制在正文前设计“证据展板” (Exhibits)，文字内容必须为视觉逻辑服务，而非单纯补充。
3. **零噪声约束**: 严禁在正文中使用加粗格式（靠逻辑和结构体现重点），禁止平庸的黑话堆砌。
