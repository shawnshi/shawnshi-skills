# 团标捕手 (tuanbiaodownloader)

针对特有的中国特定“团体标准”展示库进行深度文件穿透以及页面影像批量自动提取、复合成卷的专门型智能下载脚手架程序。

## 核心能力
- **URL 的智能解析重组**：自动化辨别例如 ttbz.org.cn 特定的加密阅览层，强硬解析取得底部的基础影像传输路径参数。
- **长时效防护拦截支持**：自身配置容灾恢复断续抓取监控功能，保证大基数页面抓取过程不断线。
- **最终合成转化**：收集到的图像碎片可按最高清晰度直接交并合成为带有查阅目录属性的正规 PDF 出版级档案文档。

## 使用场景
在进行“中国医疗设备合规与政策壁垒调查”以及对国内特定协会发行的指导手册进行信息囤积提取（如提取 ttbz.org.cn 等）的情况下强制配置并执行。并可组合 `document-summarizer` 与 `research-analyst` 调用。
