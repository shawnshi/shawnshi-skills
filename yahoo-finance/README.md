# 雅虎金融数据引擎 (yahoo-finance)

全球金融市场数据的确定性访问工具。支持股价追溯、基本面核查、新闻聚合，输出结构化 JSON 或 Rich 表格。

## 核心能力
- **市场数据**: 历史 OHLCV + 机构级统计（最大回撤、涨跌幅、波动率），支持日内K线间隔（`1m` ~ `1mo`）。
- **基本面分析**: 深度聚焦核心因子（市值、PE、**ROE**、**PB**），可选 `--full-info` 获取全量数据。
- **LLM 防噪设计**: `--lean` 模式截断冗长 K 线，极速压缩 Context Token，专注 Summary。
- **新闻聚合**: 关联特定 Ticker 的最新商业新闻。
- **智能解析**: 模糊输入"苹果"自动解析为 `AAPL`，已知 Ticker 直连跳过搜索 API。
- **工程健壮性**: 内置超时控制、指数退避重试、结构化错误报告、标准退出码。
- **投研诊断**: 结合新增专属 `stock_analyzer` Agent 身份，实现一键穿透数据迷雾，输出机构级个股诊断报告。

## 使用场景
- "诊断某公司过去一年的回撤风险与基本面健康度"
- "对比多家公司的 ROE 与 PB 等核心交易因子"
- "获取近期与该公司相关的新闻"
- "分析过去5天的1小时K线数据"
- "@stock_analyzer 请对特斯拉出具一份投资价值与诊断研报"
