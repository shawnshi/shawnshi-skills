# 雅虎金融数据引擎 (yahoo-finance)

全球金融市场数据的确定性访问工具。支持股价追溯、基本面核查、新闻聚合，输出结构化 JSON 或 Rich 表格。

## 核心能力
- **市场数据**: 历史 OHLCV + 自动摘要统计（涨跌幅、均价、波动率），支持日内K线间隔（`1m` ~ `1mo`）。
- **基本面分析**: 精简输出核心字段（市值、PE、行业等），可选 `--full-info` 获取全量数据。
- **新闻聚合**: 关联特定 Ticker 的最新商业新闻。
- **智能解析**: 模糊输入"苹果"自动解析为 `AAPL`，已知 Ticker 直连跳过搜索 API。
- **工程健壮性**: 内置超时控制、指数退避重试、结构化错误报告、标准退出码。

## 使用场景
- "查看某公司过去一年的股价走势"
- "对比多家公司的基本面数据"
- "获取近期与该公司相关的新闻"
- "分析过去5天的1小时K线数据"
