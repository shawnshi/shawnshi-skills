# 第一性原理 (First Principles)

基于埃隆·马斯克的物理学思维框架的基础推理方法论。将问题解构为最基本的事实真理，而非通过类比进行表面推理。

## 核心能力
- **解构 (Deconstruct)**：将复杂问题拆解为最微小、最纯粹的基础组成部分。
- **挑战 (Challenge)**：系统性地质疑和打破现有的假设与经验主义的软性约束。
- **重构 (Reconstruct)**：完全基于坚实的基础真理与客观的硬性约束，从头构建创新解决方案。

## 使用场景
当用户明确要求使用“第一性原理”进行分析，或面临需要打破思维定势、寻求根本性逻辑突破的复杂问题时调用此技能。
