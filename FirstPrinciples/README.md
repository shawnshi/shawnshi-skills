> **核心内核**：剥离类比思维的干扰，将复杂问题拆解为不可还原的物理事实，重新构建最优解。

## 0. 本质与边界 (Essence & Boundary)
- **核心定义**: 根本性推理引擎，通过挑战所有既定假设，从第一性原理出发重建逻辑。
- **反向定义**: 它不是一种改进现有方案的方法，而是对方案进行彻底“强拆”并重建的工具。
- **费曼比喻**: 就像是你不是在考虑如何改进旧的公文包，而是问“为什么我们需要公文包？我们要移动的是什么？”，最后你可能发明了轮子。

## 1. 生态位映射 (Ecosystem DNA)
- **MSL 契约**: 处理“公理”、“硬约束”、“软假设”等底层逻辑实体。
- **ACE 角色**: 作为 **Logic Auditor (逻辑审计者)**，负责清理系统中的“认知冗余”。

## 2. 逻辑机制 (Mechanism)
- [Deconstruct] -> [Challenge Constraints] -> [Identify Hard Truths] -> [Reconstruct Optima]

## 3. 策略协议 (Strategic Protocols)
- **物理性优先**：真实约束必须源于物理或现实，而非政策、传统或竞争对手的行为。
- **推倒重来 (Fresh Start)**：当核心假设被证伪时，必须停止修补，启动全局重建流程。
