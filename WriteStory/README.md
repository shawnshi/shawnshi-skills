> **核心内核**：基于 Will Storr 科学的 7 维叙事引擎。将故事从单一维度升华为“性格缺陷-主题辩论-逻辑闭环”的深度生命体。

## 0. 本质与边界 (Essence & Boundary)
- **核心定义**: 虚构创作的精密编译器，负责将碎片化的创意组合成符合认知心理学的叙事结构。
- **反向定义**: 它不是一个扩写 AI，而是一个具备“叙事科学”指导的共创伙伴。
- **费曼比喻**: 就像是给你请了一个好莱坞金牌编剧当顾问，他不仅帮你写，还会告诉你为什么这一场戏必须这样安排才能抓住观众的心。

## 1. 生态位映射 (Ecosystem DNA)
- **MSL 契约**: 处理“性格缺陷”、“救赎弧线”、“神秘信息管理”等叙事语义实体。
- **ACE 角色**: 作为 **Narrative Producer (叙事生产者)**。

## 2. 逻辑机制 (Mechanism)
- [Interview & Extraction] -> [Bible Building (7 Layers)] -> [Chapter Milestone Mapping] -> [Multi-Pass Drafting]

## 3. 策略协议 (Strategic Protocols)
- **七层耦合架构**：任何故事必须同步映射意义、性格、情节、神秘、世界、关系、散文七个维度。
- **新鲜度强制审计**：内置反陈词滥调系统，自动识别并拦截通用的 AI 剧情套路。
