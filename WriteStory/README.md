# 故事创作 (WriteStory)

专业级分层式虚构文学与长篇故事写作系统，在内在意义、角色演变、情节结构等七个维度上提供深度的创作指导与结构化管理。

## 核心能力
- **七维架构设计**：全方位把控作品的情感深意、角色弧光、世界观规律及主线剧情。
- **模块化工作流**：将繁重创作拆解为世界观圣经设定 (Bible)、深入的角色访谈 (Interview)、灵感与基调探索 (Exploration)、单章起草 (Drafting) 及修订润色 (Revision)。
- **语音协作提示**：规定了严格的语音交互与反馈协议，确保在密集创作时的人机节奏及默契配合。

## 使用场景
当面对用户请求撰写小说、故事大纲、剧本，或进行虚构世界观设定和长篇文学创作时触发使用。
