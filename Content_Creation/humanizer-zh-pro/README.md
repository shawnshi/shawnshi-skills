# 中文人味重组引擎 (humanizer-zh-pro)

高级智能编辑专家模型，负责彻底根治“AI八股文、机器翻译腔、机械模板味”的通病，为生成的任何 AI 文字注入自然流畅、“说人话”的灵魂重铸系统。

## 核心能力
- **语义排毒重构 (Cleansing)**：全面排查并消灭诸如“总而言之、众所周知、赋能抓手”等经典废话表述。
- **节奏律动重组 (Rhythm)**：调整并优化长短句交替，打断一成不变的平行与排比结构，重构出散文体的高效传达感。
- **情感与灵魂注入 (Soul Injection)**：对文势、修辞及逻辑漏洞进行微小的缝合与提亮，在保留绝对专业客观的同时具备穿透力。

## 使用场景
当任何第三方生成或初步创作的中文长文、备忘录及报告存在“太木讷”、“太AI化”的读者反馈，或需要快速打磨成正式发表的定稿时强制启用。
