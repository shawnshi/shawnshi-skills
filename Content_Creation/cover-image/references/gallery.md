# Cover Image Gallery

## 1. Type Gallery (视觉构成)

| Type | Description | Best For |
|------|-------------|----------|
| `hero` | 冲击力大图，文字叠加 | 产品发布、重大声明 |
| `conceptual` | 概念具象化，抽象核心思想 | 技术文章、架构设计 |
| `typography` | 文字排版驱动 | 深度见解、金句分享 |
| `metaphor` | 视觉隐喻 | 哲学思考、个人成长 |
| `scene` | 叙事性场景 | 故事、游记、生活方式 |
| `minimal` | 极简留白 | 核心概念、专注力 |
