# Domain: Military Strategy
# Source: Sun Tzu, Carl von Clausewitz, J.F.C. Fuller, John Boyd
# Structural_Primitives: maneuver, concentration, logistics, friction, fog_of_war, ooda

## Core Objects
- **Force**: 军事力量，包括兵力、火力、信息力等多维要素
- **Terrain**: 战场的地理与信息环境，对攻防具有决定性影响
- **Command**: 指挥控制系统，决定决策质量与执行效率
- **Logistics**: 后勤保障，物资、弹药、人员的持续供应
- **Intelligence**: 情报信息，降低不确定性、揭示敌方意图

## Core Morphisms
- **Maneuver**: 通过机动获得位置优势，以空间换时间
- **Concentration**: 在决定性点集中优势力量形成局部压倒
- **Fire and Movement**: 火力压制与机动推进的协同
- **Combined Arms**: 多兵种协同作战，产生互补效应
- **Strategic Depth**: 利用空间纵深消耗、迟滞、反击敌人

## Theorems / Patterns

### 1. OODA Loop
**内容**: 观察(Observe) → 判断(Orient) → 决策(Decide) → 行动(Act) 的快速循环，通过更快的节奏打乱对手决策

**Applicable_Structure**: 竞争性环境中需要快速响应和适应的场景

**Mapping_Hint**: 映射到"市场竞争中的快速迭代"、"产品开发的敏捷方法"、"危机公关的响应速度"

**Case_Study**: 海湾战争中，美军通过信息化指挥系统实现OODA循环速度远超伊军，形成决策优势

### 2. Schwerpunkt (Center of Gravity)
**内容**: 集中力量攻击敌方最关键的能力或意志来源，而非分散兵力全面进攻

**Applicable_Structure**: 资源有限时需要选择关键突破口的战略决策

**Mapping_Hint**: 映射到"产品核心功能的单点突破"、"营销资源的集中投放"、"竞争中的差异化定位"

**Case_Study**: 诺曼底登陆选择诺曼底而非加莱，虽防御较弱但成功建立桥头堡后切断德军西线补给

### 3. Asymmetric Warfare
**内容**: 弱势方通过非传统手段（游击、恐怖、网络）攻击强势方的弱点，改变传统力量对比

**Applicable_Structure**: 资源劣势方挑战市场领导者的竞争策略

**Mapping_Hint**: 映射到"初创公司挑战行业巨头"、"颠覆性创新"、"蓝海战略"、"去中介化"

**Case_Study**: 越南战争中，越共利用地形和人民支持开展游击战，抵消美军技术优势

### 4. Fog of War
**内容**: 战争中的不确定性、信息不对称、情报不完整，导致计划与执行永远存在偏差

**Applicable_Structure**: 任何充满不确定性的竞争环境

**Mapping_Hint**: 映射到"市场预测的不确定性"、"竞争情报的局限性"、"计划经济的谬误"、"敏捷战略的必然性"

**Case_Study**: 拿破仑远征俄国时，严重低估了地理、气候和后勤困难，计划迅速失效

### 5. Defensive Depth and Elastic Defense
**内容**: 通过纵深防御消耗敌人进攻动能，在敌方力量衰减时反击，而非在边境硬碰硬

**Applicable_Structure**: 面对强势竞争者的防御策略

**Mapping_Hint**: 映射到"市场防御中的撤退与反击"、"产品线的梯次防御"、"损失核心用户后的挽回策略"

**Case_Study**: 苏军在东线战场的纵深防御，通过空间换取时间，消耗德军突击力量后发起反攻
