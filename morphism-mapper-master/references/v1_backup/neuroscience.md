# Domain: Neuroscience
# Source: Santiago Ramón y Cajal, Donald Hebb, Eric Kandel, John O'Keefe
# Structural_Primitives: neuron, synapse, plasticity, network, oscillation

## Core Objects
- **Neuron**: 神经系统的基本功能单元，通过电化学信号接收、处理和传递信息
- **Synapse**: 神经元之间的连接点，通过神经递质实现信号传递
- **Neural Network**: 相互连接的神经元群体，形成功能性信息处理回路
- **Action Potential**: 神经元产生的全或无电脉冲，长距离信息编码方式
- **Neurotransmitter**: 突触间传递信号的化学信使（多巴胺、血清素、谷氨酸等）

## Core Morphisms
- **Synaptic Transmission**: 电信号 → 化学信号 → 电信号的跨突触传递
- **Plasticity (LTP/LTD)**: 突触强度的长时程增强或抑制，学习的细胞基础
- **Integration**: 神经元将数千个输入信号加权求和，决定是否发放
- **Pattern Recognition**: 神经网络从输入中提取特征、分类和预测
- **Oscillation**: 神经元群体的节律性同步活动，协调信息处理

## Theorems / Patterns

### 1. Hebb's Rule
**内容**: 一起激活的神经元连接在一起（Cells that fire together, wire together），同时活跃的突触被强化

**Applicable_Structure**: 关联学习、习惯形成、同时发生事件的联结记忆

**Mapping_Hint**: 映射到"品牌与场景的关联营销"、"用户使用习惯的培养"、"团队成员的协作默契"

**Case_Study**: 经典条件反射中，巴甫洛夫的狗学会将铃声（条件刺激）与食物（非条件刺激）关联

### 2. Predictive Coding
**内容**: 大脑不是被动接收感官输入，而是主动预测感官输入，仅传递预测误差

**Applicable_Structure**: 基于模型减少信息处理负荷的高效系统

**Mapping_Hint**: 映射到"产品设计的用户预期管理"、"默认设置减少决策负担"、" surprise 作为营销工具"

**Case_Study**: 视觉系统对移动物体的平滑追踪依赖于预测模型，而非逐帧处理

### 3. Dopamine Reward Prediction Error
**内容**: 多巴胺神经元编码奖励预测误差（实际奖励 - 预期奖励），而非奖励本身，驱动学习和动机

**Applicable_Structure**: 需要持续激励和学习的系统

**Mapping_Hint**: 映射到"游戏化设计的奖励机制"、"员工激励的预期管理"、"产品更新的惊喜感营造"

**Case_Study**: 赌博成瘾部分源于多巴胺系统对间歇性奖励的过度反应，而非奖励本身

### 4. Neural Darwinism
**内容**: 神经系统通过过度产生突触连接，然后选择性保留和修剪，实现适应性优化

**Applicable_Structure**: 从多样性中通过选择产生优化的系统

**Mapping_Hint**: 映射到"初创公司的快速试错"、"产品功能的迭代优化"、"组织变革的渐进式调整"

**Case_Study**: 婴幼儿期大脑突触密度达到峰值，随后根据经验选择性修剪，形成高效网络

### 5. Embodied Cognition
**内容**: 认知不仅发生在脑中，而是大脑、身体与环境相互作用的涌现属性

**Applicable_Structure**: 需要理解认知与环境、身体紧密耦合的系统

**Mapping_Hint**: 映射到"用户体验的环境依赖"、"线下体验对品牌认知的影响"、"具身化的产品设计"

**Case_Study**: 手持热咖啡的人比手持冷咖啡的人更倾向于评价他人为"温暖"，身体感觉影响社会判断
