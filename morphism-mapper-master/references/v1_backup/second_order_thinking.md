# Domain: Second-Order Thinking
# Source: Howard Marks, Charlie Munger, Michael Mauboussin
# Structural_Primitives: first_order, second_order, feedback_loop, unintended_consequences, time_delay

## Core Objects
- **First-Order Effect**: 行动的直接、即时、可见的后果
- **Second-Order Effect**: 行动间接、延迟、连锁的后果，常被忽视但影响更大
- **Feedback Loop**: 系统输出作为输入影响系统自身，形成正反馈或负反馈
- **Time Delay**: 原因与结果之间的时间滞后，导致决策与效果的错位
- **Hidden Variable**: 未被直接观察但对结果有重大影响的因素

## Core Morphisms
- **Consequence Propagation**: 初始行动 → 第一阶后果 → 第二阶后果 → 系统性影响
- **Feedback Activation**: 系统输出被感知并反馈，放大或抑制初始变化
- **Time Lag**: 决策输入与效果显现之间的时间间隔，导致学习延迟
- **Counter-Adaptation**: 系统参与者对其他参与者行动的适应性反应
- **Emergent Pattern**: 个体理性决策聚合产生的集体非理性结果

## Theorems / Patterns

### 1. Cobra Effect (Perverse Incentive)
**内容**: 针对问题的直接解决方案反而产生与目标相反的后果，因为忽视了二阶效应

**Applicable_Structure**: 政策设计、激励机制、系统干预

**Mapping_Hint**: 映射到"销售激励导致的短视行为"、"算法优化产生的意外后果"、"监管套利"

**Case_Study**: 英属印度悬赏眼镜蛇皮导致民众养殖眼镜蛇，取消赏金后养殖蛇被放生，蛇患比原来更严重

### 2. Goodhart's Law
**内容**: 当指标成为目标时，它就不再是一个好的指标，因为人们会优化指标而非真实目标

**Applicable_Structure**: KPI设计、绩效评估、算法目标函数

**Mapping_Hint**: 映射到"KPI扭曲行为"、"点击量驱动的内容低俗化"、"考试导向的教育异化"

**Case_Study**: 银行以贷款数量考核员工，导致大量低质量贷款和随后的坏账危机

### 3. Feedback Delay Trap
**内容**: 系统反馈的时间延迟导致决策者误判行动效果，产生过度调整或方向错误

**Applicable_Structure**: 具有显著时间滞后的复杂系统

**Mapping_Hint**: 映射到"营销投入与销售的时差"、"人才培养的长周期"、"技术投资的回报滞后"

**Case_Study**: 淋浴水温调节中，延迟感知导致反复过度调整，难以稳定

### 4. Tragedy of the Commons
**内容**: 个体理性利用共享资源导致集体非理性结果，资源被过度消耗

**Applicable_Structure**: 共享资源管理、公共品供给、负外部性

**Mapping_Hint**: 映射到"开源社区的资源枯竭"、"公共数据滥采"、"共享经济的公地悲剧"

**Case_Study**: 公地放牧中，每个牧民增加牲畜的私人收益大于分摊的公共损失，导致过度放牧和草场退化

### 5. Reflexivity
**内容**: 市场参与者的认知影响市场，市场变化又影响认知，形成自我强化的反馈循环

**Applicable_Structure**: 预期影响现实的金融市场、社会现象

**Mapping_Hint**: 映射到"股价与基本面的相互影响"、"品牌认知的自我实现"、"舆论与现实的互动"

**Case_Study**: 索罗斯的反身性理论解释泡沫形成：上涨预期推动买入，买入推动上涨，强化预期直至崩溃
