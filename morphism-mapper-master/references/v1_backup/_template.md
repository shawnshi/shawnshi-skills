## 使用说明

此模板用于创建新的 Domain B 领域文件。

命名规范：
- 文件名使用小写 + 下划线，如 `quantum_mechanics.md`
- 领域名称首字母大写，如 "Quantum Mechanics"

内容要求：
1. 每个 Object 和 Morphism 必须有清晰定义
2. 每个 Theorem 必须包含 Applicable_Structure 和 Mapping_Hint
3. Tags 用于帮助系统匹配最适合的 Domain B
