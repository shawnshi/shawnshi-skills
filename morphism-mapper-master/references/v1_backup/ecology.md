# Domain: 生态学
# Source: Odum《生态学基础》, Levin《脆弱的领地》, Meadows《系统之美》
# Structural_Primitives: 种群、共生、承载力、演替、食物网、生态位

## Core Objects
- 种群: 同种生物在特定区域的集合
- 群落: 不同种群在特定区域的集合
- 生态系统: 生物群落与其物理环境的相互作用
- 生态位: 物种在生态系统中的功能位置和角色
- 栖息地: 物种生活的物理环境
- 承载力: 环境能持续支持的最大种群数量

## Core Morphisms
- 竞争: 对有限资源的争夺
- 捕食: 一种生物以另一种生物为食
- 共生: 不同物种间的长期密切关系（互利、偏利、寄生）
- 演替: 群落随时间的有序变化
- 能量流动: 通过食物链的能量传递
- 物质循环: 营养物质在生物与非生物间的循环

## Theorems / Patterns

### 1. 承载力与种群调节
**内容**: 种群增长受环境承载力限制。当种群接近承载力时，密度制约因素（资源竞争、疾病）会抑制增长，形成 S 型增长曲线。

**Applicable_Structure**: 增长极限、市场饱和、资源约束、自我调节

**Mapping_Hint**: 当 Domain A 追求增长时，识别承载力。超过承载力的增长会导致崩溃，可持续的策略是在承载力范围内优化。

**Case_Study**: 鹿群在缺乏捕食者时过度繁殖，最终因食物短缺而大规模死亡；产品的市场渗透极限。

### 2. 关键种与基石种
**内容**: 某些物种对生态系统的影响远大于其生物量比例。移除关键种会导致生态系统结构的剧烈变化。

**Applicable_Structure**: 关键角色、杠杆点、系统依赖、影响力不均

**Mapping_Hint**: 当 Domain A 是复杂系统时，识别"关键种"——投入少量资源保护或激活它们，可以产生系统性影响。

**Case_Study**: 海獭控制海胆数量，保护海藻林；星状海 Pisaster 是经典的关键种，移除后生物多样性急剧下降。

### 3. 演替与顶极群落
**内容**: 生态系统经历从裸岩到复杂群落的可预测演替序列，最终达到相对稳定的顶极群落。干扰可以重置演替。

**Applicable_Structure**: 发展阶段、成熟度、干扰与更新、长期趋势

**Mapping_Hint**: 当 Domain A 处于发展阶段时，识别所处演替阶段。不同阶段的策略不同：早期阶段追求定植，晚期阶段追求稳定。

**Case_Study**: 森林火灾后的恢复；产品的市场生命周期；组织的成长阶段。

### 4. 营养级联效应
**内容**: 食物链中某一层级的变化会通过级联效应影响多个其他层级。顶级捕食者的消失会导致整个生态系统的重组。

**Applicable_Structure**: 连锁反应、系统关联、间接影响、层级管理

**Mapping_Hint**: 当 Domain A 是多层级系统时，考虑营养级联——改变一个层级可能产生意想不到的远距离效应。

**Case_Study**: 黄石公园重新引入狼群，不仅控制了鹿群，还改变了河流走向（因为植被恢复减少了侵蚀）。

### 5. 边缘效应与生态交错带
**内容**: 两种生态系统的交界处（生态交错带）往往具有最高的生物多样性和生产力，因为结合了两种系统的特征。

**Applicable_Structure**: 跨界创新、边界价值、多样性、融合

**Mapping_Hint**: 当 Domain A 处于两个领域的交界处时，认识到这是创新热点——边缘位置提供了独特的资源和机会。

**Case_Study**: 森林与草原交界处的物种丰富度最高；跨学科研究往往产生突破性发现；处于不同文化交界处的人群更具创造力。
