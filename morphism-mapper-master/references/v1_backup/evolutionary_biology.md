# Domain: 进化生物学
# Source: Darwin《物种起源》, Dawkins《自私的基因》
# Structural_Primitives: 选择、适应、变异、遗传、竞争、共生

## Core Objects
- 种群: 同种生物的集合，进化的基本单位
- 基因: 遗传信息的基本单位，自然选择的作用对象
- 环境: 选择压力来源，包括物理环境和生物环境
- 适应度: 个体在特定环境中生存和繁殖的能力
- 生态位: 物种在生态系统中的功能位置
- 突变: 遗传信息的随机改变，变异的来源

## Core Morphisms
- 自然选择: 环境对适应度的筛选，适者生存
- 遗传: 基因从亲代到子代的传递
- 变异: 基因突变和重组产生多样性
- 竞争: 对有限资源的争夺
- 共生: 不同物种间的互利关系
- 漂变: 随机因素导致的基因频率变化

## Theorems / Patterns

### 1. 红皇后效应
**内容**: "在这个国度，你必须不断奔跑才能保持在原地。" 物种必须不断进化以维持相对适应度，因为竞争对手和捕食者也在进化。

**Applicable_Structure**: 零和竞争、军备竞赛、相对优势 vs 绝对进步

**Mapping_Hint**: 当 Domain A 处于激烈竞争环境时，强调相对优势的重要性。绝对能力的提升可能被竞争对手的同步提升抵消，必须持续迭代。

**Case_Study**: 宿主与寄生虫的协同进化，双方都在进化，但相对优势保持不变。

### 2. 间断平衡
**内容**: 进化不是渐进的，而是长期稳定与短期快速变化的交替。物种在大多数时间保持稳定，偶尔经历快速分化。

**Applicable_Structure**: 渐进优化 vs 范式跃迁、稳定期 vs 爆发期、量变 vs 质变

**Mapping_Hint**: 当 Domain A 长期优化却收效甚微时，考虑是否存在"物种形成事件"的可能——打破现有平衡，进入新的稳定态。

**Case_Study**: 化石记录显示物种形态长期稳定，地质短时期内发生快速变化，而非达尔文预期的渐进变化。

### 3. 进化稳定策略 (ESS)
**内容**: 在一个种群中，如果大多数成员采用某种策略，任何偏离该策略的突变策略都无法获得更高收益，该策略就是进化稳定的。

**Applicable_Structure**: 均衡状态、策略选择、偏离惩罚、系统稳定性

**Mapping_Hint**: 当 Domain A 存在多种策略选择时，寻找 ESS——即使不是最优，但一旦形成就难以被打破的均衡点。

**Case_Study**: 鹰鸽博弈中，种群会收敛到鹰鸽混合的稳定比例，纯鹰或纯鸽都不是 ESS。

### 4. 亲缘选择
**内容**: 个体不仅追求自身繁殖成功，也会帮助遗传相关个体繁殖。利他行为的进化基础是共享基因。

**Applicable_Structure**: 自私 vs 利他、个体 vs 群体、成本收益计算

**Mapping_Hint**: 当 Domain A 存在合作与竞争的矛盾时，引入"亲缘系数"概念。看似利他的行为，可能是对"扩展自我"的投资。

**Case_Study**: 蜜蜂工蜂不繁殖，而是帮助蜂后繁殖，因为它们与蜂后的后代共享更多基因。

### 5. 适应度景观
**内容**: 适应度可以可视化为一个多峰景观，种群通过自然选择爬坡。但局部最优可能阻碍达到全局最优。

**Applicable_Structure**: 局部最优陷阱、探索 vs 利用、创新者的困境

**Mapping_Hint**: 当 Domain A 陷入局部最优时，需要"降低适应度"的变异来跳出当前峰值，探索新的可能。

**Case_Study**: 种群被困在局部最优峰，必须通过遗传漂变或环境变化"下坡"，才能到达更高的山峰。
