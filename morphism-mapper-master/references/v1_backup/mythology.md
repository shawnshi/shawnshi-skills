# Domain: 神话学/原型心理学
# Source: Campbell《千面英雄》, Jung《原型与集体无意识》, Eliade《神圣与世俗》
# Structural_Primitives: 原型、英雄之旅、转化、阈限、神圣、叙事

## Core Objects
- 英雄: 故事的主角，代表自我或集体
- 导师: 提供智慧和工具的引路人
- 阴影: 被压抑的、需要整合的黑暗面
- 门槛: 从已知世界进入未知世界的边界
- 圣杯: 追求的目标，象征完整或救赎
- 原型: 集体无意识中的普遍模式

## Core Morphisms
- 启程: 从平凡世界进入冒险
- 试炼: 面对挑战和考验
- 转化: 经历本质的改变
- 回归: 带着获得的智慧返回
- 牺牲: 放弃旧我以获得新生
- 启示: 获得深刻的洞察或领悟

## Theorems / Patterns

### 1. 英雄之旅的单一神话
**内容**: 所有文化的英雄故事都遵循同一基本结构：启程（分离）→ 启蒙（考验与转化）→ 回归（带着礼物返回）。

**Applicable_Structure**: 用户旅程、产品体验、个人成长、组织变革

**Mapping_Hint**: 当 Domain A 涉及用户体验或变革过程时，将其设计为英雄之旅——识别平凡世界、召唤、导师、试炼、最终回归的完整弧线。

**Case_Study**: 星球大战、狮子王、哈利波特都遵循这一结构；用户从"意识到问题"到"解决问题"的旅程也可映射。

### 2. 阈限空间 (Liminal Space)
**内容**: 仪式和转化的核心阶段是"阈限"——处于两种状态之间的模糊地带，旧身份已死，新身份未生。

**Applicable_Structure**: 过渡期、身份转换、不确定性、创造性空间

**Mapping_Hint**: 当 Domain A 涉及用户或组织的转变时，设计"阈限体验"——不是立即跳到新状态，而是允许在模糊空间中探索和学习。

**Case_Study**: 成年礼中的隔离期；旅行中的机场、酒店等"非此非彼"空间；产品试用期的探索阶段。

### 3. 阴影整合
**内容**: 个体或文化中被压抑、否认的部分（阴影）会持续以破坏性方式出现，直到被承认和整合。

**Applicable_Structure**: 负面反馈、被忽视的需求、黑暗模式、完整性

**Mapping_Hint**: 当 Domain A 存在反复出现的问题或负面行为时，检查是否是被压抑的"阴影"——不是对抗，而是整合和转化。

**Case_Study**: 品牌忽视的用户抱怨最终爆发为公关危机；个人压抑的情绪最终以疾病形式出现。

### 4. 永恒回归与仪式时间
**内容**: 神话时间不是线性进展，而是循环的。通过仪式，可以周期性回到原初时间，更新世界的活力。

**Applicable_Structure**: 仪式感、周期性更新、节奏设计、仪式感

**Mapping_Hint**: 当 Domain A 感觉疲惫或失去活力时，引入"仪式"——不是持续线性推进，而是周期性回到起点，重新激活意义。

**Case_Study**: 新年的意义；产品的大版本发布作为"仪式"；团队的定期复盘和重启。

### 5.  trickster 的颠覆力量
**内容**: Trickster（骗子/小丑）是规则的打破者，通过颠覆和混乱带来新可能。它挑战既定秩序，暴露虚伪。

**Applicable_Structure**: 创新突破、规则挑战、认知颠覆、混乱中的秩序

**Mapping_Hint**: 当 Domain A 陷入僵化时，引入 trickster 能量——不是渐进改进，而是通过看似破坏性的行为打破僵局，创造新空间。

**Case_Study**: 硅谷的"颠覆者"；愚人节营销；那些通过打破行业规则而成功的公司。
