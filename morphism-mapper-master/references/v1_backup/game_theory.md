# Domain: 博弈论
# Source: von Neumann & Morgenstern《博弈论与经济行为》, Nash, Schelling《冲突的策略》
# Structural_Primitives: 策略、收益、均衡、信息、承诺、威胁

## Core Objects
- 玩家: 博弈中的决策主体
- 策略: 玩家可选择的行动方案
- 收益: 策略组合下各玩家的得失
- 信息集: 玩家知道什么、不知道什么
- 均衡: 稳定的策略组合，没有玩家愿意单方面偏离
- 共同知识: 所有玩家都知道，且知道其他人都知道...

## Core Morphisms
- 策略选择: 根据对其他玩家行为的预期选择策略
- 收益计算: 评估不同策略组合的结果
- 均衡收敛: 通过学习和调整达到稳定状态
- 信号传递: 通过行动传递私有信息
- 承诺: 限制未来选择以影响他人预期
- 威胁与承诺: 改变他人收益结构以影响其行为

## Theorems / Patterns

### 1. 纳什均衡
**内容**: 在非合作博弈中，如果每个玩家的策略都是对其他玩家策略的最优反应，则该策略组合构成纳什均衡。没有玩家可以通过单方面改变策略而获得更高收益。

**Applicable_Structure**: 稳定状态、相互依赖、策略互动、预期协调

**Mapping_Hint**: 当 Domain A 涉及多方互动时，寻找纳什均衡——即使不是最优，但一旦形成就难以打破的稳定点。

**Case_Study**: 囚徒困境中的（背叛，背叛）；价格战中所有厂商都降价；交通中所有人都靠右行驶。

### 2. 囚徒困境与集体行动问题
**内容**: 个体理性选择（背叛）导致集体非最优结果。合作对整体有利，但个体有动机搭便车或背叛。

**Applicable_Structure**: 公地悲剧、合作困境、信任建立、激励机制

**Mapping_Hint**: 当 Domain A 面临合作困境时，认识到个体理性与集体利益的冲突。解决路径：重复博弈、改变收益结构、引入惩罚机制。

**Case_Study**: 环境污染、过度捕捞、价格战、公共品供给不足。

### 3. 承诺的价值
**内容**: 通过限制自己的未来选择（承诺），可以改变他人的预期，从而获得更好的结果。可信的承诺比灵活性更有价值。

**Applicable_Structure**: 战略互动、预期管理、可信承诺、先发优势

**Mapping_Hint**: 当 Domain A 需要影响他人行为时，考虑做出可信承诺——展示"别无选择"的决心，迫使对方调整策略。

**Case_Study**:  Cortés 烧毁船只向士兵展示必胜决心；企业宣布不降价承诺以阻止竞争对手降价。

### 4. 混合策略与不可预测性
**内容**: 在某些博弈中，最优策略是随机化选择，使对手无法预测。可预测性会被对手利用。

**Applicable_Structure**: 竞争策略、不可预测性、随机化、欺骗

**Mapping_Hint**: 当 Domain A 的对手在利用你的可预测性时，引入随机化——让对手无法确定你的下一步，从而无法针对性应对。

**Case_Study**: 足球点球中射门方向的最优策略是随机化；军事战略中的佯攻和欺骗；扑克中的虚张声势。

### 5. 焦点效应 (Schelling Point)
**内容**: 在缺乏沟通的多人协调博弈中，某些解决方案因其"显著性"而成为自然的协调点，即使它们没有内在优势。

**Applicable_Structure**: 协调问题、共识形成、显著性、文化默认

**Mapping_Hint**: 当 Domain A 需要多方协调但无法充分沟通时，设计"显著"的解决方案——利用共同文化、历史或逻辑上的独特性作为协调锚点。

**Case_Study**: 在纽约与陌生人约定见面，大多数人选择中央车站或时报广场；产品标准的确立往往依赖于先发优势或显著性。
