# Domain: 网络理论
# Source: Barabási《链接》, Watts《六度分隔》, Newman《网络科学》
# Structural_Primitives: 节点、连接、度、路径、集群、中心性

## Core Objects
- 节点: 网络中的基本单元，可以是个人、网页、神经元等
- 边: 节点间的连接，可以是有向或无向，加权或非加权
- 度: 节点连接的边数，衡量节点的连接性
- 路径: 从一个节点到另一个节点的路线
- 组件: 相互连接的节点集合
- 社区: 内部连接密集、外部连接稀疏的节点群

## Core Morphisms
- 连接: 建立节点间的关系
- 传播: 信息、影响或疾病在网络中的流动
- 级联: 一个节点的状态变化引发连锁反应
- 聚类: 节点形成紧密连接的群体
- 渗流: 网络中 giant component 的形成过程
- 同步: 节点状态的协调一致

## Theorems / Patterns

### 1. 小世界效应
**内容**: 大多数真实网络的平均路径长度很短，即使网络非常稀疏。"六度分隔"现象。

**Applicable_Structure**: 信息传播、关系链、远距离连接、世界压缩

**Mapping_Hint**: 当 Domain A 涉及信息或影响传播时，利用小世界特性——通过少数关键连接，可以触达整个网络。

**Case_Study**: Milgram 实验：随机选择的两个人平均只需要 6 个中间人就能建立联系。

### 2. 无标度网络与优先连接
**内容**: 真实网络中，少数节点拥有大量连接（枢纽），多数节点只有少量连接。新节点倾向于连接已经高度连接的节点（富者愈富）。

**Applicable_Structure**: 马太效应、资源集中、枢纽依赖、幂律分布

**Mapping_Hint**: 当 Domain A 呈现高度不均分布时，识别和保护枢纽节点。攻击枢纽可以迅速瓦解网络，保护枢纽则增强鲁棒性。

**Case_Study**: 互联网链接分布、社交网络中的意见领袖、航空网络的枢纽机场。

### 3. 网络鲁棒性与脆弱性
**内容**: 无标度网络对随机攻击具有鲁棒性（攻击普通节点影响不大），但对针对性攻击（攻击枢纽）非常脆弱。

**Applicable_Structure**: 系统韧性、单点故障、攻击策略、防御设计

**Mapping_Hint**: 当 Domain A 需要增强鲁棒性时，避免过度依赖少数枢纽；当需要攻击对手时，集中资源打击其枢纽。

**Case_Study**: 互联网可以承受大量随机路由器故障，但攻击关键 DNS 服务器可以瘫痪大片网络。

### 4. 弱连接的力量
**内容**: 弱连接（不频繁的、远距离的关系）在信息传播中比强连接更重要，因为它们连接不同的社区，带来新信息。

**Applicable_Structure**: 信息获取、创新来源、社交资本、桥接作用

**Mapping_Hint**: 当 Domain A 面临信息茧房或创新枯竭时，投资于弱连接——它们比强连接更能带来新机会和不同视角。

**Case_Study**: Granovetter 的研究发现，大多数人通过弱连接找到工作，而非亲密朋友。

### 5. 网络效应与梅特卡夫定律
**内容**: 网络的价值与节点数的平方成正比。每增加一个节点，都为所有现有节点增加价值。

**Applicable_Structure**: 平台经济、用户增长、临界规模、正反馈循环

**Mapping_Hint**: 当 Domain A 具有网络效应时，优先追求用户增长而非短期盈利。达到临界规模后，网络会自我强化。

**Case_Study**: 电话网络、社交网络、支付网络的价值随用户数量指数增长。
