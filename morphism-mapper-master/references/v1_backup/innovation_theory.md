# Domain: Innovation Theory
# Source: Joseph Schumpeter, Clayton Christensen, Eric von Hippel, Jeffery Immelt
# Structural_Primitives: creative_destruction, disruptive_innovation, s_curve, diffusion, paradigm_shift

## Core Objects
- **Innovation**: 被成功商业化或实施的新想法、产品、流程或商业模式
- **Technology S-Curve**: 技术性能随时间和投入呈S形增长的发展轨迹
- **Dominant Design**: 在特定产品类别中被广泛接受的标准架构
- **Adopter Categories**: 创新者、早期采用者、早期大众、晚期大众、落后者
- **Innovation Pipeline**: 从想法到商业化的多阶段筛选和发展流程

## Core Morphisms
- **Creative Destruction**: 新创新摧毁旧产业结构和既有企业竞争优势
- **Technology Diffusion**: 创新从起源向更广泛用户群体的传播过程
- **Crossing the Chasm**: 从早期市场（技术爱好者）向主流市场（实用主义者）的跨越
- **Architectural Innovation**: 改变组件连接方式但保留核心技术的创新
- **Radical Innovation**: 打破既有技术轨迹、开辟新市场的突破性创新

## Theorems / Patterns

### 1. Disruptive Innovation
**内容**: 低端或新市场进入者以更简单、更便宜的产品起步，逐步改进最终颠覆在位企业

**Applicable_Structure**: 在位企业面临被新进入者颠覆风险的市场

**Mapping_Hint**: 映射到"初创公司挑战行业巨头"、"产品简化的市场机会"、"过度服务客户的风险"

**Case_Study**: 数字摄影从低端 hobbyist 市场起步，性能逐步提升最终取代胶片巨头柯达

### 2. Innovator's Dilemma
**内容**: 成功的在位企业理性地投资于维持性创新（服务现有客户），错过颠覆性创新机会

**Applicable_Structure**: 成功企业面临技术范式转换时的战略困境

**Mapping_Hint**: 映射到"核心业务与创新业务的资源分配"、"现有客户的束缚"、"独立业务单元的必要性"

**Case_Study**: 尽管发明了数字相机，柯达因担心冲击胶片业务而未能成功转型

### 3. S-Curve Dynamics
**内容**: 技术性能初期增长缓慢，中期快速改进，后期接近物理极限增长放缓

**Applicable_Structure**: 技术投资决策和时机选择

**Mapping_Hint**: 映射到"技术投资的进入时机"、"旧技术的退出决策"、"研发投入的边际收益"

**Case_Study**: 摩尔定律接近硅基芯片物理极限，促使行业探索量子计算等新范式

### 4. Lead User Innovation
**内容**: 创新常由处于市场前沿、需求超前于主流的用户（领先用户）创造，而非企业内部研发

**Applicable_Structure**: 识别和捕获前沿用户需求的产品开发

**Mapping_Hint**: 映射到"高端用户作为创新来源"、"社区驱动的产品开发"、"开源生态"

**Case_Study**: 极限运动装备多由运动员自行改装后商业化，而非品牌原创

### 5. Lock-in and Network Effects
**内容**: 用户规模增加带来价值提升（直接/间接网络效应），形成转换成本锁定用户

**Applicable_Structure**: 平台型业务和生态系统的竞争

**Mapping_Hint**: 映射到"平台战略的飞轮效应"、"数据网络效应"、"生态系统锁定"、"多归属成本"

**Case_Study**: QWERTY键盘和Windows操作系统通过网络效应和转换成本维持主导地位
