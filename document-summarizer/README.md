# 文档摘要与合规审计 (document-summarizer)

针对诸如医疗文档在内的大型、复杂或合规性要求极高的异构文档，进行高通量的精准聚合梳理及合规排查的提炼系统。

## 核心能力
- **批处理提取归纳**：拥有自动识别提取、智能摘要凝练的长文档降维分析能力。
- **定制化标签注入**：允许依照战略分析要求，将具有强信息密度的元数据 (Metadata) 和自定标签无缝注入最终报告。
- **合规性审计扫描**：可配置专门针对特定垂直行业要求（合规、敏感数据脱敏等）的排查脚本协同调度。

## 使用场景
当需要一次性阅读、通篇精简或审核数百页的长文件、行业政策、医疗白皮书或是大批量文档时。
