# 战略情报枢纽 (intelligence-hub)

V4.0 版本的复合战略情报搜集与处理指挥中心，专门负责在高难度行业与高竞争度的企业对标战中实施多源聚合情报抓取。

## 核心能力
- **五阶段情报循环协议**：标准集成由对齐校准、多维侦察、AI 精炼推演、红队对抗审计至情报简报生成的全生命周期作战机制。
- **动态战略校准**：基于 `memory.md` 词频分析自动调整关键词权重，确保情报捕获始终对齐当前项目重心。
- **信息熵严格控制**：强制约束简报输出在 Top 10 条目与 L1-L4 分级框架内，拒绝对事实的过度解读与主观猜测。
- **高可用采集引擎**：100+ RSS/API 源并发抓取，支持增量去重缓存、指数退避重试与代理配置。

## 使用场景
受托搜寻与竞争对手极深度的发展内幕、绘制多维度技术与市场交锋沙盘以及撰写具有战略高度的决策简报前夕启用。

## 依赖安装
```bash
pip install -r requirements.txt
```
