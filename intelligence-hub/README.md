# 战略情报枢纽 (intelligence-hub)

V3.2 版本的复合战略情报搜集与处理指挥中心，专门负责在高难度行业与高竞争度的企业对标战中实施多源聚合情报抓取。

## 核心能力
- **五阶段情报循环协议**：标准集成由对齐校准、多维侦察、降噪提炼、红队对抗审计至情报简报生成的全生命周期作战机制。
- **信息熵严格控制**：强制约束简报输出在有限长度与特定颗粒度内，拒绝对事实的过度解读与主观猜测。
- **专用自动化指令**：关联了专用的 Python 收集调度脚手架以突破常规查询限制。

## 使用场景
受托搜寻与竞争对手极深度的发展内幕、绘制多维度技术与市场交锋沙盘以及撰写具有战略高度的决策简报前夕启用。
