# Intelligence Hub (战略情报作战中枢)

## Vision
构建具备“战略嗅觉”的多源情报引擎。通过解耦表现层与逻辑层，实现对全球泛技术趋势与垂直医疗 IT 信号的高密度、自动化捕获与推演。

## Index
- `SKILL.md`: 核心 SOP 与 Thought-Refiner 协同协议。
- `scripts/fetch_news.py`: [V4.0] 解耦版抓取引擎（支持多线程与模板渲染）。
- `scripts/update_index.py`: 情报总线索引维护器。
- `references/strategic_focus.json`: 动态战略关键词与评分权重配置。
- `references/briefing_template.md`: 简报表现层 Markdown 模板。
