# Intelligence Hub V3.2 (战略情报作战中枢)

## Vision
构建具备“战略嗅觉”的多源情报引擎。通过解耦采集（Sensor）与推演（Deduction），实现对全球泛技术趋势与垂直医疗 IT 信号的高密度、自动化捕获与“So What?”级精炼。V3.2 引入了动态战略校准、增量抓取与博弈审计协议。

## Index
- `SKILL.md`: 核心 SOP V3.2，含动态校准、博弈审计与物理索引协议。
- `scripts/calibrate_focus.py`: [NEW] 基于 `memory.md` 自动校准情报关键词权重。
- `scripts/fetch_news.py`: [IMPROVED] 支持增量抓取（Cache）与代理配置。
- `scripts/update_index.py`: [IMPROVED] 同时维护 Markdown 总目与 JSON 检索索引。
- `references/quality_standard.md`: IQS 情报质量准则（L1-L4 分级）。
- `references/karpathy_feeds.json`: 90 个全球深度技术订阅源。
- `references/strategic_focus.json`: 动态战略关键词配置（自动生成）。
- `references/briefing_template.md`: 咨询级简报 Markdown 模板。
