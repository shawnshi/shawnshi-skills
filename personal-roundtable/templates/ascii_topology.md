# ASCII Topology Templates for Synthesis

## 1. The 2x2 Tension Matrix
```text
      Impact/Urgency
           ^
           |   (High Tension)
           |   [A] <-----> [B]
           |
-----------+-----------> Strategic Importance
           |
           |
           |   (Low Tension)
           |
```

## 2. The OODA/Feedback Loop
```text
  [Observe] -----> [Orient]
      ^               |
      |               v
  [Act] <--------- [Decide]
```

## 3. The Iceberg Model (Causal Layers)
```text
   (Events)         [Visible Result]
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~ (Waterline)
   (Patterns)       [Historical Trend]
   (Structures)     [Systemic Design]
   (Mental Models)  [Core Beliefs]
```

## 4. The Causal Loop Diagram (CLD)
```text
   [A] --- (+) ---> [B]
    ^                |
    |                v
   [D] <--- (-) --- [C]
```
