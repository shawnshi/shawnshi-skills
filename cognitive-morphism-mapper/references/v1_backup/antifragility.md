# Domain: Antifragility
# Source: Nassim Nicholas Taleb
# Structural_Primitives: antifragile, fragile, robust, hormesis, optionality

## Core Objects
- **Fragile**: 受波动性和压力伤害的系统，需要平静环境
- **Robust**: 抵抗冲击、保持不变的系统，不受波动影响但也不受益
- **Antifragile**: 从波动、随机性、压力中获益、成长壮大的系统
- **Hormesis**: 低剂量毒物刺激效应，适度压力增强系统能力
- **Optionality**: 拥有选择权利而非义务，在不确定性中保留上行收益、限制下行损失

## Core Morphisms
- **Stress Adaptation**: 压力 → 恢复 → 超补偿（比原来更强）
- **Via Negativa**: 通过移除而非添加来改进，减法优于加法
- **Skin in the Game**: 决策者承担后果，利益与风险对齐
- **Barbell Strategy**: 极端安全 + 极端冒险的组合，避免中间地带
- **Convexity**: 从波动中获益的非线性收益结构

## Theorems / Patterns

### 1. Antifragility via Overcompensation
**内容**: 系统在应对压力时往往过度修复，修复后的能力超过受损前水平（超补偿原理）

**Applicable_Structure**: 需要建立韧性并从挑战中学习的系统

**Mapping_Hint**: 映射到"危机后的组织变革"、"竞争压力驱动的创新"、"适度授权培养员工能力"

**Case_Study**: 肌肉在承受超过日常负荷的锻炼后，修复过程中变得更强壮（hypertrophy）

### 2. Barbell Strategy
**内容**: 将大部分资源放在极端安全（避免毁灭风险），小部分放在极端高风险高回报（捕获不确定性的收益）

**Applicable_Structure**: 面对不确定性的资产配置和战略选择

**Mapping_Hint**: 映射到"核心业务保守 + 创新业务激进"、"现金牛产品 + 探索性投资"、"职业发展的稳定+冒险组合"

**Case_Study**: 投资中将90%配置于国债等安全资产，10%配置于高风险高回报的机会，而非全部中等风险投资

### 3. Via Negativa (Subtractive Knowledge)
**内容**: 知道什么是错的（减法）比知道什么是对的（加法）更可靠，避免伤害比寻求收益更有效

**Applicable_Structure**: 风险管理和决策优化

**Mapping_Hint**: 映射到"产品功能减法"、"负面清单管理"、"删除无效流程"、"避免认知偏差的清单"

**Case_Study**: 医学中，减少错误（洗手、检查清单）比寻找治愈方法更能挽救生命

### 4. Convexity and Optionality
**内容**: 面对不确定性时，拥有选择权（optionality）的结构让损失有限、收益无限，从不确定性中获益

**Applicable_Structure**: 需要管理不确定性并保留灵活性的场景

**Mapping_Hint**: 映射到"战略期权投资"、"模块化架构保留切换自由"、"轻资产商业模式"、"人才期权池"

**Case_Study**: 风险投资组合中，单个项目损失上限为投资额，但成功项目收益无上限，整体具有凸性

### 5. Lindy Effect
**内容**: 非易腐事物的预期寿命与其已存在时间成正比，存在越久越可能继续存在

**Applicable_Structure**: 技术、思想、书籍等文化产品的生命力预测

**Mapping_Hint**: 映射到"经典商业模式的持久性"、"品牌资产的时间价值"、"技术选型的成熟度考量"

**Case_Study**: 已存在100年的技术（如轮子、刀叉）预期还能存在100年，而刚出现的新技术可能很快被淘汰
