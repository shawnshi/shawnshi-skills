# 营销心理学库 (marketing-psychology)

沉淀了 70 余个心理学原则、心智模型和行为科学理论的营销知识宝库，专门用于解答“人们为什么会这样买单”的核心疑虑并指导转化构建。

## 核心能力
- **基础认知解构**：运用第一性原理 (First Principles)、乔布斯待办理论 (JTBD) 和逆向思考等帮助重塑战略思考。
- **消费者行为操控与偏误**：囊括单纯曝光效应、禀赋效应、沉没成本谬误及选择悖论，帮助你在产品展示中操纵其预期。
- **商业化转化与定价模型**：深入解析诱饵效应 (Decoy Effect)、锚定效应 (Anchoring)、损失厌恶和魅力定价法以优化产品转化漏斗。

## 使用场景
用户提到“心理学”、“心智模型”、“认知偏误”、“行为科学”、“销售为什么不佳”以及“如何促进决策与购买”时，必须引入相应的心理学模型框架。
