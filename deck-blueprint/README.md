## 1. 核心定位 (Core Identity)
**Deck Blueprint** 是一个集成了麦肯锡（McKinsey）、波士顿（BCG）和贝恩（Bain）生产管线的“战略视觉架构师”。它不生产平庸的 PPT，它通过高强度的逻辑重构与视觉隐喻，帮助用户在复杂博弈中达成共识。

## 2. 顶级咨询管线 (The MBB Pipeline)

该技能严格执行五阶段工业级生产流程：

1.  **Ghost Decking (战略幽灵草案)**: 
    - 在填充内容前，先锁定 30 个**叙事性标题 (Lead-ins)**。
    - **水平逻辑检查**: 仅读标题即可构成完整故事。
2.  **Evidence-Mesh (证据网构建)**: 
    - 强制对每一个断言进行溯源。标注为 `[Source]` 或 `[Hypothesis]`。
    - 消除 AI 幻觉，确保方案在法律和合规层面鲁棒。
3.  **Blueprint Execution (精密设计)**: 
    - 依据 **金字塔原理** 组织页面内容。
    - 生成 `<STYLE_INSTRUCTIONS>` 视觉契约。
4.  **Adversarial Audit (红队对抗审计)**: 
    - 模拟“怀疑论决策者”和“保守派执行者”进行压力测试。
    - 强制自我纠偏，提升方案在现实工作流中的兼容性。
5.  **Physical Manifestation (物理转化)**: 
    - 联动 `slide-deck` 技能，实现从 Markdown 蓝图到物理 `.pptx` 文件的闭环。

## 3. 核心律令 (The Commandments)

*   **Lead-in 优先**: 严禁使用描述性标题（如“项目背景”），必须使用结论性主题句（如“由于 A 因素，B 市场正面临结构性衰退”）。
*   **MECE 原则**: 逻辑架构必须做到“相互独立，完全穷尽”。
*   **拒绝 AI 废话**: 严禁空洞的过渡句和通用的占位符。
*   **战略指纹**: 自动注入用户在 `memory.md` 中的核心行业主张（如 MSL, ACE, HITL）。

## 4. 目录结构 (File Structure)

*   `SKILL.md`: 技能逻辑与工作流定义。
*   `TEMPLATE.md`: 咨询级单页幻灯片结构字典。
*   `_DIR_META.md`: 目录元数据与版本历史。
*   `README.md`: 战略操作手册。

在 Gemini CLI 中输入：

系统将首先启动 **第一阶段：战略透视**，并向您确认 Ghost Title List。
