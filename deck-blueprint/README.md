- **核心定义**: 战略演示文稿的底层逻辑引擎，负责构建“ Lead-in 驱动”的叙事结构。
- **反向定义**: 它不是一个 PPT 模板，而是一个思维架构工具。
- **费曼比喻**: 它像是在盖大楼前画的蓝图，即使没有贴瓷砖（还没排版），你也能从图纸上看出这栋楼（这套方案）能不能稳固支撑。

- **Lead-in 准则**：每一页标题必须是结论性的主题句，能够独立构成叙事逻辑。
