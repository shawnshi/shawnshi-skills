# Dark Terminal Style Reference

Hacker-centric, low-light aesthetic optimized for GitHub READMEs and technical blogs.

## Page Style
- **Background**: `#0f0f1a` (Obsidian)
- **Grid**: Invisible
- **Font**: `SF Mono` or `Fira Code`

## Node Styles (Overrides)

| Role | Fill Color | Stroke Color | Font Color |
|------|------------|--------------|------------|
| **Base** | `#1a1a2e` | `#00FF00` | `#00FF00` |
| **Core** | `#16213e` | `#00FF00" | `#00FF00` |
| **Active** | `#0f3460` | `#00FF00" | `#00FF00` |
| **Alert** | `#533483` | `#FF0000" | `#FF0000` |
| **External** | `#1a1a2e` | `#E94560" | `#E94560` |

## Edge Style
- **Stroke**: `#00FF00` (Neon Green)
- **Style**: `edgeStyle=orthogonalEdgeStyle;rounded=1;` (Rounded corners)
