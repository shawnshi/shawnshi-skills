# Blueprint Style Reference

Professional engineering aesthetic with high contrast and grid-aligned components.

## Page Style
- **Background**: `#0a1628` (Deep Midnight Blue)
- **Grid**: Visible, Cyan
- **Font**: `Courier New` (Monospace)

## Node Styles (Overrides)

| Role | Fill Color | Stroke Color | Font Color |
|------|------------|--------------|------------|
| **Base** | `#0d2137` | `#00FFFF` | `#FFFFFF` |
| **Core** | `#162d42` | `#00FFFF` | `#FFFFFF` |
| **Active** | `#1c3a52` | `#00FFFF` | `#FFFFFF` |
| **Alert** | `#3d1a1a` | `#FF0000` | `#FFFFFF` |
| **External** | `#2a1a3d` | `#BC80FF` | `#FFFFFF` |

## Edge Style
- **Stroke**: `#00FFFF` (Cyan)
- **Style**: `edgeStyle=orthogonalEdgeStyle;rounded=0;` (Sharp corners)
