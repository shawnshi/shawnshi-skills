# 医疗行业雷达 (Healthcare Industry Radar)

## 技能概述

**医疗行业雷达** 是为医疗信息化 (HIT) 领域的高管与战略咨询专家打造的进阶、战略级 Gemini 技能。它不同于传统的、充满噪音的新闻聚合器，而是采用高度结构化的情报检索与分析框架。

它专门针对精心挑选的国内外 HIT 巨头（如 Epic, Oracle, 卫宁健康, 东软, 医渡云等），并通过三大核心战略透视镜（资本生态、底层产品线重构、政企级市场收割）来过滤他们近期的动态。

最终的产出不是简单的摘要拼凑，而是一份极具实战价值的战略简报。它强制采用 S-T-C (Signal-Threat-Countermeasure) 框架，剥离公关废话，直接输出具有可行性的商业洞察。

## 核心特性

- **防拒载触发设计 (Anti-Undertriggering):** 技能的底层设计确保了即使用户只是模糊地询问 HIT 竞争对手的动向或市场动态，也能主动且精准地唤醒雷达，防止遗漏关键战略情报。
- **渐进式加载架构 (Progressive Disclosure):** 保持主技能工作流的极致精简，同时将深度的评价标准（如 S-T-C 推演细则和目标阵营列表）卸载到独立的参考文档中。
- **S-T-C 高阶战略推演框架:**
  - **核心信号去水 (Signal):** 萃取最原始、客观的商业动作，撕开企业 PR 的包装。
  - **护城河穿透透视 (Threat):** 宏观上评估该动作是否踩中了政策杠杆（如 DRG/DIP、互联互通等），微观上评估其对现有护城河的直接威胁或降维打击程度。
  - **实战反击指令 (Countermeasure):** 拒绝正确的废话。强制要求给出具体的防御或进攻建议（例如：防卫性并购、推进特定子产品落地、结盟“超级算力玩家”等）。
- **最高预警与情报清单 (Executive Summary & Data Table):** 自动生成开篇的“摘要预警”以及高密度的 Markdown 表格（关键资讯清单），便于在深度阅读前快速扫描全局。
- **严格时效约束:** 强制将搜索半径限定在“本周以内”，以保持极高的信噪比。若无重大战略异动，宁可留白也不凑字数。

- “执行医疗行业雷达”
